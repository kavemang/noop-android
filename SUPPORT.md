# Support

Use GitHub issues for NOOP Android bugs, feature requests, and support questions:

<https://github.com/kavemang/noop-android/issues>

For Android bugs, include:

- NOOP Android version
- Phone model
- Android version
- Strap model and firmware, if known
- What you expected to happen
- What actually happened
- A redacted diagnostic report or strap log when relevant

NOOP Android is an independent, unofficial project. It is not affiliated with, endorsed by, or connected to WHOOP, Inc., and it is not a medical device.
